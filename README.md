## Dev

``` bash
# install dependencies
$ npm install # Or yarn install

# serve with hot reload at localhost:3000
$ npm run dev

# generate static project
$ npm run generate
```

These docs are built with [Nuxt](https://github.com/nuxt/nuxt.js)
